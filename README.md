Player Control 
A robot is being controlled
The player controls a third-person robot.
Movement is controlled using W, A, S, and D
Camera movement is controlled with the mouse
The player attacks using the left mouse button
The player can block incoming damage using a block input using "Q"
Character movement, rotation, and animation are handled through Unity’s Third Person Controller
Open doors using "E"

Basic Gameplay
Goals / Objectives
The main objective of Circuit Breakers is to survive enemy encounters by defeating enemies while maintaining the player’s health or collecting energy orbs to advance to the next level.

Gameplay Explanation
The player spawns into a level filled with hostile enemies and must engage in combat by attacking them while avoiding or blocking incoming damage. Throughout the level, the player can collect energy orbs and defeat enemimes to aid progression. The player’s health is displayed on the screen and decreases as damage is taken.

Game Ending Conditions
The game ends when the player’s health reaches zero or the timer reaches zero
When this occurs, player movement and combat are disabled
A death message is displayed on the screen to indicate game over
